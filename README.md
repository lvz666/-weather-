天气预警推送服务
> 这是一个基于 Node.js 的轻量级气象预警轮询检测工具，专为 iOS 用户提供极简的天气预警推送服务。
> 功能
>  * 定时轮询：自动检测气象预警信息。
>  * 实时推送：支持将预警信息推送到 Bark 等通知服务。
> 如何运行
>  * 克隆仓库：git clone https://github.com/你的用户名/仓库名.git
>  * 安装依赖：进入目录后，运行 npm install。
>  * 配置环境：在目录下创建 config.json 并填入你的推送密钥（请妥善保管）。
>  * 启动服务：运行 node server.js。
