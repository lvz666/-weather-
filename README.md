# 书呆子-八月-两周

iOS 极端/平常天气预警推送服务。

## 功能
- 实现对天气预警信息的轮询检测。
- (这里填上你的具体推送方式，比如推送到哪里)

## 如何运行
1. 安装 Node.js 环境。
2. 运行 `npm install` 安装依赖。
3. 运行 `node server.js` 启动服务。

⁠server.js⁠：项目的核心代码，负责定时轮询气象预警接口，解析预警数据，并将信息推送到目标终端。

index.html⁠：项目的前端界面，用于直观查看当前的运行状态及设置。

subscriptions.json⁠：存储用户预警订阅信息的配置文件。
